# E105 – Rejestr Piratów Drogowych

Projekt semestralny

Prosta aplikacja webowa umożliwiająca zarządzanie rejestrem piratów drogowych.  
Pozwala na dodawanie, edytowanie, usuwanie oraz przeglądanie wpisów w bazie danych.

---

## Funkcjonalności

Aplikacja obsługuje pełen zestaw operacji CRUD:

- Dodawanie piratów drogowych  
- Edytowanie istniejących wpisów  
- Usuwanie wpisów  
- Wyświetlanie listy piratów drogowych  

---

## Technologie

Projekt został wykonany z użyciem następujących technologii:

- Node.js  
- Express  
- EJS  
- MongoDB  

---
## Instrukcja instalacji i uruchomienia

**Klonowanie repozytorium:**

    git clone https://github.com/7kez7/E105
    cd E105

**Instalacja zależności projektu:**

    npm install

**Uruchomienie aplikacji:**

    npm start

Aplikacja będzie dostępna pod adresem:  
http://localhost:7000

---

## Lista endpointów

- `/` – Lista piratów drogowych  
- `/add` – Dodawanie pirata drogowego  
- `/stats` – Statystyki  
- `/about` – O nas
