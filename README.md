# E105 – Rejestr Piratów Drogowych

Projekt semestralny

Prosta aplikacja webowa umożliwiająca zarządzanie rejestrem piratów drogowych.  
Pozwala na dodawanie, edytowanie, usuwanie oraz przeglądanie wpisów w bazie danych.

---

## Funkcjonalności

Aplikacja obsługuje pełen zestaw operacji CRUD:

- Dodawanie piratów drogowych  
- Edytowanie istniejących wpisów  
- Usuwanie wpisów  
- Wyświetlanie listy piratów drogowych  
- Podgląd statystyk  

---

## Technologie

Projekt został wykonany z użyciem następujących technologii:

- Node.js  
- Express  
- EJS  
- MongoDB
